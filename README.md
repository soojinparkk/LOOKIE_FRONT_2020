  ex) 

    1) MainActivity - A,B,C 버튼 생성
        - A버튼 클릭 시 AActivity로 이동
        - B버튼 클릭 시 BActivity로 이동
        - C버튼 클릭 시 CActivity로 이동

    4) CActivity - 뒤로 가기 버튼 생성 → 클릭 시 MainActivity로 이동
